# PLC 사전 테스트 프로그램 – 사용자 여정

## 페르소나
이름: 김엔지니어  
직무: 장비업체 PLC 엔지니어  
목표: 현장 반입 전 MES ↔ PLC 통신을 완벽히 검증해 현장 디버깅 시간을 절반으로 줄인다.

## 1. 설치 & 최초 실행
1. 이메일로 전달받은 설치파일 Setup.exe 더블클릭
2. "다음→설치" 클릭, 10초 이내 설치 완료
3. 첫 실행 시 업데이트 체크 후 Main 화면 표시

## 2. Main 화면 – 장비 & 시나리오 타입 선택
1. 드롭다운에서 대상 장비 모델 선택
2. Common / Normal / Abnormal 3개 버튼 중 ‘Normal’ 클릭
3. ‘연결 설정’ 버튼 클릭 → 모달에서 PLC IP/Port 입력 후 ‘연결 테스트’
   • 연결 성공: 녹색 체크표시 → ‘확인’

## 3. 시나리오 목록 화면
1. Normal 시나리오 1~5가 카드 형태로 나열
2. ‘시나리오 2’ 카드 클릭

## 4. 시나리오 상세 & 테스트 실행
1. 시나리오 설명, 예상 이벤트 흐름 표 확인 후 ‘테스트 시작’ 클릭
2. 실시간 이벤트 대시보드에 신호 스트림 그래프 표시
3. 각 이벤트 완료 시 리스트에 Pass/Fail 아이콘 갱신
4. 모든 이벤트 끝나면 상단에 ‘테스트 완료 – Pass 10 / Fail 0’ 배너

## 5. 결과 확인 & 리포트
1. ‘리포트 다운로드’ 클릭 → PDF, Excel 저장
2. ‘로그 업로드’ 클릭 → ZIP 자동 생성 후 서버 전송, 완료 토스트 메시지

## 6. 추가 테스트 / 종료
1. ‘목록으로 돌아가기’ 클릭해 다른 시나리오 반복 테스트
2. 작업 끝나면 창 닫기 → 프로그램 자동 업데이트 체크 후 종료

## 주요 감정곡선
• 설치&연결 성공 시 ‘안도’ → 자동 테스트 진행 중 ‘편리함’ → Pass 결과 확인 시 ‘만족’ → 리포트·로그 자동 업로드로 ‘업무 단축’
